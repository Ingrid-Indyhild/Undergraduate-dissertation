# Automated Testing Framework for Educational Web Projects (Cypress)

## Description
This repository contains a testing framework developed as part of a qualification work.

The main goal of the project is to automate the validation of educational web projects by checking:
- HTML structure
- CSS styles
- JavaScript functionality

The framework is built using Cypress and is designed to simplify and standardize the evaluation process.

–––––––––––––––––––––––––––––––––––––––––––––––––––––––––––––––––––––

## Objectives
- Reduce manual effort in checking student web projects
- Ensure consistent evaluation criteria
- Detect typical errors automatically
- Improve feedback speed

---

## Technologies
- JavaScript
- Node.js
- Cypress
